# YGO Hand Probability Simulator

A Yu-Gi-Oh opening hand probability simulator built with Materialize CSS.

**[Live Demo](https://arsidee.github.io/Hand-Probability/)**

## Features

- Build your deck by adding cards and setting quantities (1–3 copies)
- Select which cards you need in your opening hand
- Set minimum copies required per card
- Toggle between going first (5 cards) or going second (6 cards)
- Runs 100,000 simulated hands to calculate probability
